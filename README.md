## What this repository is

A research programme investigating the hypothesis that institutional reform with persistent multi-domain legitimacy is bounded below by a *legitimacy-transfer generation* (~25–40 years), and that current technological developments operate on connectivity and knowledge-store reconciliation rather than on the legitimacy-transfer process itself.

This is descriptive scientific work on observed coordination timescales. The repository inherits the **non-prescription clause** (see `CONVENTIONS.md`): minimum coordination timescales are observed phenomena, not endorsed as correct, optimal, or politically legitimate. **A floor is not a target.**

## What this repository is not

- It is not a Coastline. No external endorsement is asserted.
- It is not a policy recommendation. Extracted prescriptive use is unauthorised.
- It is not yet evidentiary. All hypotheses are pre-falsification with binding timelines.

## Core hypotheses (frozen v0.2)

**H₀ — Legitimacy-Transfer-Generation Hypothesis.** Institutional reform with persistent multi-domain legitimacy is bounded below by the minimum interval over which an institution's decision-making cohort can be fully replaced and acculturated to recognised authority. Typically L₅-scale, ~25–40 years. Has not substantially shortened across recorded history.

**H₁ — Pressure Corollary.** Current technological developments impose external demand rates exceeding the H₀ floor. Signature is second-order: increased variance, increased reversal frequency, oscillation, silent error propagation — *not* uniform speed-up.

**M — Mechanism Conjecture.** To first order, on observed historical scales, technology acts on connectivity between agents and on reconciliation rate of knowledge stores. It does not act on the legitimacy-transfer process itself. The L₅ floor reflects the un-technologised half of the coordination problem.

Each hypothesis carries explicit falsification handles and binding timelines. See `meta/falsification-tracker.md`.
