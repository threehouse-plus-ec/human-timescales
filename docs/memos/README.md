## Discipline

Memos are versioned (v0.1, v0.2, ...). Each carries:

- Endorsement Marker (provisional unless explicitly upgraded).
- Inheritance from the canonical scoping document's non-prescription clause.
- Stance log.
- Falsification handles where applicable.

## When a memo graduates

- Upward to a Sail (`../sails/`) when its argument is ready for cross-referenced essay form.
- Upward to a scoping revision (`../scoping/`) when its findings warrant integration into the canonical scoping document.
- To Breakwater (`../ledger-drafts/`) when its central claim is ready for ledger entry.
- Archived if superseded or weakened by later evidence.
