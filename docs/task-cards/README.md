- ID, Title, Type, Status.
- Drafter / Reviewer stances.
- Parent / Depends on / Feeds.
- Deliverable.
- Success criterion.
- Falsification handle (where applicable).
- Estimated scope.
- Risks / open questions.

Status values: PROPOSED · AUTHORISED · IN-PROGRESS · COMPLETE · BLOCKED · DEFERRED.
