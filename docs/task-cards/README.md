# Task Cards

Operational task cards specifying the programme's empirical pilots, methodological infrastructure, and synthesis deliverables.

## Current set

**v0.1** — `human-timescales-task-cards-v0.1.md`. Eleven cards (TC-01 through TC-11).

## Critical path

TC-06/TC-07 → TC-01 → TC-09, with TC-04 and TC-05 contributing in parallel. TC-02/TC-03 parallel-run for T4 kill-criterion satisfaction.

## Discipline

Each task card declares:
- ID, Title, Type, Status.
- Drafter / Reviewer stances.
- Parent / Depends on / Feeds.
- Deliverable.
- Success criterion.
- Falsification handle (where applicable).
- Estimated scope.
- Risks / open questions.

Status values: PROPOSED · AUTHORISED · IN-PROGRESS · COMPLETE · BLOCKED · DEFERRED.
